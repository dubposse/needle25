# Needle25

Needle25 ist eine Web-App zur Verwaltung deiner Musiksammlung, Wunschliste und persönlichen Charts.

## Features

- Musiksammlung (CRUD)
- Wunschliste
- Persönliche Charts (öffentlich)
- Charts nach Genre entdecken
- Benutzerauthentifizierung

## Tech Stack

- Next.js (App Router)
- PostgreSQL (Neon)
- JavaScript (kein TypeScript)

## Live

https://needle25.vercel.app

## Ziel

Dieses Projekt wurde entwickelt, um Full-Stack-Entwicklung mit Fokus auf Backend-Systeme, Authentifizierung und Datenbankdesign zu lernen.
