# Needle25
[![License: AGPL v3](https://img.shields.io)](https://opensource.org)

Needle25 is a web app for managing your music collection, wishlist, and personal charts.

## Features

- Music collection (CRUD)
- Wishlist
- Personal charts (public)
- Discover charts by genre
- User authentication

## Tech Stack

- Next.js (App Router)
- PostgreSQL (Neon)
- JavaScript (no TypeScript)

## Live

https://needle25.vercel.app

## Overview

Needle25 was designed and developed as a full-stack application with a focus on structured data handling, authentication, and database integration. The project emphasizes practical functionality and a clear separation between frontend, backend, and data logic.
